# Salish Sea Dreaming — Project Vision

A generative art installation exploring the ecological imagination of the Salish Sea bioregion through AI-mediated visual art.

## What This Is

Salish Sea Dreaming uses StyleGAN2 and diffusion models trained on bioregional imagery to generate living visual compositions. The project bridges ecological data, indigenous place-knowledge, and computational art to create immersive installations that invite viewers into relationship with the more-than-human world of the Salish Sea.

## Core Components

- **StyleGAN2 pipeline**: latent space exploration of marine and coastal imagery
- **LoRA style transfer**: Briony Penn-inspired watercolor aesthetic applied to generated outputs
- **StreamDiffusion + TouchDesigner**: real-time generative installation framework
- **Ecological data integration**: herring populations, tidal patterns, seasonal cycles as generative inputs

## Design Principles

- Art in service of ecological awareness, not spectacle
- Bioregional grounding — every training image and data source is place-specific
- Iterative: gallery exhibitions inform the next generation of the pipeline
- Accessible: web and physical installation formats
