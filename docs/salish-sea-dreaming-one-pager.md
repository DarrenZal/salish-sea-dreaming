# The Salish Sea Dreaming — Concept Brief

**For:** Raf (curator, Digital Ecologies / Salt Spring) | **Drafted by:** Darren Zal | **For review by:** Pravin Pillay, Carol Anne Hilton

---

## Vision

The Salish Sea Dreaming is an immersive installation that creates a co-dreaming space — holding both aspiration and the nightmare of ecological loss. Audiences don't just witness; they participate in the dreaming, processing anxiety and hope together through an experience grounded in Indigenous cosmology and real-time ecological data from the Salish Sea.

This is not a purely idealistic piece. Large tankers are now visible offshore for the first time. Burrard Inlet is being dredged to double pipeline capacity. Herring balls have disappeared from Fulford Harbor since the 1910s. The installation makes this visible while offering the ontological shift Carol Anne describes: passing through the mandala gate where wisdom is internally generated, not externally imposed.

## Core Concept: Co-Dreaming

**Aspiration + nightmare held together.** The seven domains of Indigenous worldview order structure the experience. The installation facilitates a self-generated ontological shift — the settler misunderstanding frames this as "giving land back" rather than becoming worthy of being welcomed. The dreaming does not impose this; it creates the conditions for it to emerge.

**Participatory, not passive.** Audience biometrics and presence influence the installation's stream of consciousness in real time, using Brad Necyk's Mind Quest architecture — a layered cognitive system (auditory, vision, emotion, memory, subconscious) that mirrors how the brain's unconscious processes shape waking thought.

## Technology Architecture

Three layers working together:

1. **Agentic World State Engine** (Brad Necyk)
   - Autonomous characters navigating a shared world with internal dialogue, actions, and emergent narrative
   - Neuroscience-grounded personality system (Big Five + wounds, defenses, epistemic boundaries, memory)
   - Character cognitive cascade: expect → perceive → feel → remember → resolve → act

2. **Knowledge Graph / Bioregional Intelligence** (Darren Zal / KOI Protocol)
   - Salish Sea ecological data: herring populations, trophic cascades, orca migration, water quality
   - The "Kwaxala" model: forage fish worth more swimming — accounting for ecosystem services
   - Bioregional knowledge network connecting the Salish Sea to other ecological intelligence nodes

3. **Living Memory Substrate** (Shawn Anderson)
   - A universal data structure where every element — image, sound, story, observation — shares a common form
   - Memories that fade if not attended to, requiring engagement to sustain — mirroring how living systems actually work
   - Community participation flows back as value to the ecosystem that generated it

## Installation Format

**Immersive LED volume** — the team has access to large-scale LED volume infrastructure, with significant GPU compute available to power real-time agentic world state, character rendering, and knowledge graph visualization. This is the scale and technical depth the piece requires to fully realize the dreaming.

## The Salish Sea Ecological Story

The dreaming is grounded in a specific ecological crisis:
- **Herring as keystone:** Forage fish that when not caught have double the value — trophic cascade supporting salmon, orcas, and the entire marine food web
- **DFO baseline problem:** The 1953 "baseline" population was already a terrible herring year — Indigenous knowledge documents a vastly different historical reality
- **Data sovereignty in action:** Heiltsuk and Wuikinuxv Nations have already challenged DFO and taken over their own stock assessment
- **The Kwaxala economic model:** "Worth more swimming" — accounting for ecological services, nutrient cycling, carbon sequestration, cultural value
