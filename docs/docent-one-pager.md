## Common symptoms

| You see | Do this |
|---------|---------|
| **Both walls black** | Wait 3 min. Still black → text Darren. (Possible Windows update overnight — known issue, they can fix remotely.) |
| **One wall black, other fine** | Text Darren with photo of which wall. Don't power-cycle the projector. |
| **Frozen image (not moving)** | Wait 2 min. Still frozen → text Darren. |
| **No sound** | Check speaker power + volume knob. Still silent → text Darren. |
| **Visitor says their dream didn't appear** | Normal — tell them "wait 30 seconds, there's a queue." If it's been 2+ min and nothing showed → text Darren. |

While talking to visitors: if a wall goes black mid-visit, smile and say **"it's self-healing, give it a moment."** Usually comes back within 2 minutes.
