What to evaluate

  1. Style transfer: Do LoRA images (#00–#07) show Briony's watercolor characteristics? (soft edges, pigment blending, ecological illustration feel)
  2. Trigger isolation: Do #08 and #09 (no brionypenn trigger, but LoRA still loaded) look different from #00–#07? If they still show Briony style, the LoRA "leaks" without the trigger.
  3. A/B delta: In Section 1, how much difference does LoRA make vs base model on the same prompt?
  4. Generalization: Does the style work on land subjects (#03–#05) or only marine (#00–#02)?
  5. Mode collapse: Do any outputs look like copies of training images rather than new compositions?
  6. Overall quality: Are these usable for the Salish Sea Dreaming installation, or do we need more training data / different hyperparameters?
